# AI News Podcast Template

이 저장소는 AI를 활용해 매일 새로운 뉴스 팟캐스트를 생성하고 배포할 수 있는 템플릿입니다.

## 핵심 컨셉: 에이전트 기반 관리
이 프로젝트는 Hermes Agent와 같은 자율 AI 에이전트가 뉴스 수집, 대본 작성, 음성 합성, RSS 피드 업데이트를 전담하도록 설계되었습니다.

### 시작하기
1. 이 저장소를 `Use this template` 버튼을 눌러 자신의 레포지토리로 복제합니다.
2. `.env` 파일을 생성하고 필요한 API 키(OpenAI, ElevenLabs 등)를 설정합니다.
3. 에이전트에게 아래 `AGENT.md` 가이드에 따라 작업을 위임합니다.
4. **필수 툴**: `notebooklm-mcp-cli`를 설치하여 소스 생성 및 연구 기능을 에이전트가 활용할 수 있게 하십시오.

---

## 🤖 에이전트 운영 가이드 (AGENT.md)

이 프로젝트는 에이전트의 자율 운영을 최우선으로 합니다. 에이전트가 작업을 수행할 때 다음 규칙을 따르도록 설정하십시오.

### 1. 주요 워크플로우
에이전트에게 다음 목표를 기반으로 주기적인 크론 잡(Cron Job)을 설정하도록 요청하세요:
- **소스 생성**: `notebooklm-mcp-cli`를 사용하여 매일 새로운 뉴스 소스를 수집하고 정리.
- **뉴스 수집**: 매일 정해진 시간에 신뢰할 수 있는 소스에서 AI 최신 뉴스를 스크래핑.
- **대본 작성**: 수집된 데이터를 바탕으로 청취자 친화적인 팟캐스트 대본(Markdown) 생성.
- **음성 합성**: ElevenLabs 등의 API를 사용하여 고품질 오디오 파일로 변환.
- **배포 및 피드 업데이트**: 생성된 오디오 파일을 호스팅하고 `feed.xml`을 업데이트하여 팟캐스트 플랫폼에 게시.

### 2. 에이전트에게 명령하는 법 (예시)
에이전트(Hermes 등)에게 다음과 같은 프롬프트를 사용하여 관리를 맡기세요:

> "현재 레포지토리의 워크플로우를 분석하고, 뉴스 수집부터 RSS 업데이트까지의 전체 파이프라인을 매일 오전 8시에 자동으로 실행하도록 크론 잡을 생성해줘."

> "새로운 오디오 파일이 생성되었습니다. `feed.xml`의 포맷을 확인하고, 최신 에피소드를 포함하도록 업데이트한 뒤 변경 사항을 커밋해줘."

### 3. 기술 스택
- **언어**: Node.js, Python (자동화 스크립트)
- **배포**: GitHub Actions (자동화), GitHub Pages (RSS 피드 호스팅)
- **오디오**: ElevenLabs API (TTS)
